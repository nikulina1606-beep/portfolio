# Портфолио Анны Никулиной

Статический адаптивный сайт без сборщика и зависимостей. Версия воссоздана по видеореференсу исходного портфолио: светлая редакционная подача, тёмный технологичный кейс SkinX и тёплый исследовательский блок. Характер дизайна сохранён, а содержание и интерактивность усилены.

## Как открыть

Откройте `index.html` в браузере.

Для локального сервера:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## Как редактировать

- Тексты и структура: `index.html`
- Стили и цвета: `styles.css`
- Галерея: `script.js`
- Изображения, PDF и видео: папка `assets`

## Что включено

- расширенный кейс SkinX с результатами и распределением ответственности;
- связка «исследование → маркетинговое решение» внутри основного кейса;
- презентационные и B2B-материалы;
- актуальное коммерческое предложение SkinX × DocMa с превью обложки;
- внутренний просмотрщик всех 10 страниц КП и отдельная кнопка скачивания PDF;
- горизонтальная галерея из 6 вертикальных AI-визуалов высокого разрешения;
- два видеоролика с аудиодорожками;
- исследовательский кейс СПбГУ и дополнительная аналитика в JASP;
- адаптивная версия с полноценным мобильным меню, анимации появления, живые показатели, активная навигация, индикатор прокрутки и лайтбокс.

## Публикация

Папку можно загрузить на GitHub Pages, Netlify, Vercel или любой обычный хостинг.
